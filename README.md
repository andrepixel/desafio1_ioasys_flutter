# Start Here

Para rodar o projeto é necessáro que você tenha, instalado no seu sistema:

* SDK Flutter
* Android Studio (Windows/Linux)
* xCode (MacOs)

Além de uma IDE para inicilizar fácil o emulador com o projeto, indico o Visual Studio Code.
